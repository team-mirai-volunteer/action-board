# 私有地ポスターマップ機能

## 概要

ボランティアが私有地に掲示したポスターの場所・枚数を地図上で報告し、市区町村単位で集計・可視化する機能。

URL: `/map/poster-residential`  
feature: `src/features/map-poster-residential/`

## ユーザーフロー

1. ナビゲーションから「私有地ポスターマップ」を選択
2. 地図が表示される（ログイン必須）
3. 地図をクリック → ピンが配置される
4. 登録フォームが表示される（住所は逆ジオコーディングで自動入力、編集可）
5. 枚数・メモを入力して登録
6. 「自分のピンを表示」トグルで自分のピンと集計マーカーを切替
7. 自分のピンをクリック → 編集・削除

## プライバシー設計

| データ | 取得 | 表示 |
|--------|------|------|
| 他ユーザーの個別位置 | **取得しない** | 表示しない |
| 市区町村集計（city_stats） | 全ユーザー分を集計 | 地図上にマーカー表示 |
| 自分のピン | 本人のみ取得可能 | トグルON時のみ表示 |
| 住所詳細 | DBに保存（RLS: 本人のみ） | フォームに「公開されません」と明示 |

### 座標の匿名化

city_stats ビューの `avg_lat`/`avg_lng` は `ROUND(AVG(lat), 2)` で小数点2桁（≒1.1km精度）に丸めている。1件のみの市区町村でも個人の正確な位置は特定不可。

## データベース

### テーブル: `residential_poster_placements`

| カラム | 型 | 説明 |
|--------|-----|------|
| id | UUID (PK) | 主キー |
| user_id | UUID (FK) | 報告者 |
| lat | DECIMAL(10,8) | 緯度 |
| lng | DECIMAL(11,8) | 経度 |
| prefecture | TEXT | 都道府県（逆ジオコーディング） |
| city | TEXT | 市区町村（逆ジオコーディング） |
| address | TEXT | 住所（ユーザー編集可） |
| postcode | TEXT | 郵便番号 |
| count | INTEGER | 掲示枚数 |
| memo | TEXT | メモ |
| mission_artifact_id | UUID (FK, nullable) | ミッション連携 |
| created_at | TIMESTAMPTZ | 作成日時 |
| updated_at | TIMESTAMPTZ | 更新日時 |

RLS: 本人のみ SELECT/INSERT/UPDATE/DELETE

### ビュー: `residential_poster_city_stats`

市区町村レベルの集計。個人情報なし。座標は匿名化済み（ROUND 2桁）。

| カラム | 説明 |
|--------|------|
| prefecture | 都道府県 |
| city | 市区町村 |
| total_count | 合計枚数 |
| placement_count | 報告件数 |
| avg_lat | 平均緯度（丸め済み） |
| avg_lng | 平均経度（丸め済み） |

## 地図のクラスタリング

- `leaflet.markercluster` を使用
- **単一の `MarkerClusterGroup`** で city stats マーカーと自分のピンを切り替える
- トグルOFF（デフォルト）: 市区町村集計マーカーを表示（ズームアウトでクラスタ化）
- トグルON: 自分のピンを表示（ズームアウトでクラスタ化）
- クラスタアイコンは `createCityStatsMarkerIcon` で統一（枚数に応じた色・サイズ）
