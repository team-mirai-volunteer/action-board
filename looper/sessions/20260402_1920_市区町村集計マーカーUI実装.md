# city-stats-markers-ui: 市区町村集計マーカーUI実装

## タスク
Milestone 3 ステップ B — ページ・クライアント・地図コンポーネント・フック・マーカーユーティリティの実装

## 作成・変更したファイル
- **新規**: `src/features/poster-placement/utils/city-stats-marker.ts` — divIcon生成ユーティリティ（サイズ・色を枚数に応じて変化）
- **編集**: `src/app/map/poster-placement/page.tsx` — `fetchCityStats`をサーバーサイドで呼び出し、initialCityStats propsとして渡す
- **編集**: `src/features/poster-placement/components/poster-placement-page-client.tsx` — useState/useCallbackでcityStats管理、登録成功後に再フェッチ
- **編集**: `src/features/poster-placement/hooks/use-poster-placement-map.ts` — onSubmitSuccessコールバックオプション追加
- **編集**: `src/features/poster-placement/components/poster-placement-map.tsx` — cityStats useEffectで集計マーカー描画・更新

## 設計判断
- 既存コードの`L.divIcon`パターン（cluster-icon.ts）に倣い、同じ`window.L`参照方式を採用
- フックのoptions引数はoptionalにして後方互換性を維持
- 集計マーカーのzIndexOffset=-1000でピン配置マーカーの下に表示
