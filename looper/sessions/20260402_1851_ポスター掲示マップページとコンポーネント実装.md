# map-page-and-components: ページ・地図コンポーネント・登録フォーム・フック実装

## タスク ID
map-page-and-components（Milestone 2 ステップ B）

## 作成したファイル
- `src/app/map/poster-placement/page.tsx` — ページ本体（認証チェック + リダイレクト）
- `src/features/poster-placement/components/poster-placement-page-client.tsx` — クライアント統合（地図 + フォーム）
- `src/features/poster-placement/components/poster-placement-map.tsx` — Leaflet 地図コンポーネント
- `src/features/poster-placement/components/placement-form.tsx` — 登録フォーム（カード型オーバーレイ）
- `src/features/poster-placement/hooks/use-poster-placement-map.ts` — 状態管理フック

## 設計判断
- 地図コンポーネントは geoman-map.tsx をベースに Geoman 関連を除去し、クリックでピン配置する簡略版として実装
- タイルレイヤーは設計ドキュメント指定の OpenStreetMap を使用
- `window.L = L` を地図初期化時に設定し、useCurrentLocation フックの依存を満たす
- フォームは useRef で枚数管理（再レンダリング抑制）、Button は shadcn/ui を利用

## 次のタスクへの申し送り
- ステップ A（server-action-and-nav）は Wave 1 で実装済み。本タスクはそれに依存する UI 側
- 地図の初期ズームは 13（設計ドキュメント指定。posting-map より広域表示）
